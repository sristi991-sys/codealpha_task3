# CodeAlpha Sudoku Solver

## Description
A Sudoku Solver implemented using Backtracking Algorithm in C++.

## Features
- User Input Puzzle
- 2D Array Representation
- Backtracking Algorithm
- Recursive Solving
- Clean Board Display

## Technologies Used
- C++
- Recursion
- Backtracking
- Arrays

## How To Run

Compile:
```bash
g++ sudoku_solver.cpp -o sudoku
```

Run:
```bash
./sudoku
```

Enter Sudoku puzzle using 0 for empty cells.

## Algorithm Used
Backtracking Algorithm
